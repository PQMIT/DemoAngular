About User Management Project

Project Overview

The User Management System is a project designed to help organizations easily manage and track user information. This system offers features such as user creation, modification, deletion, and search functionalities.

Key Features
  • User listing with pagination
  • Adding new users
  • Editing user details inline
  • Deleting users with confirmation
  • Sharing data with components
  • Filtering and sorting users
  • View Movie, Search Movie, Movie Category

Technologies Used
  • Angular 18: The core framework for building the UI and handling logic
  • PrimeNG 18: For beautiful and responsive UI components
  • PrimeFlex: CSS utilities for rapid UI design
  • LocalStorage: Temporary data storage for managing users in the browser
  • REST API: Backend service for fetching and updating user data (optional)

Developer Information

This project was developed by a team of skilled frontend developers using modern Angular and PrimeNG to deliver a responsive, efficient, and scalable user management system.
